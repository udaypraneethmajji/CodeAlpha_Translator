# CodeAlpha AI Internship - Language Translator

## 📌 Project Description
This project translates text from one language to another using Python.

## 🛠 Technologies Used
- Python
- deep-translator

## 🚀 Features
- Supports multiple languages
- Simple user input

## ▶️ How to Run
1. Install:
   pip install deep-translator

2. Run:
   python translator.py
